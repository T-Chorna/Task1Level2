Завдання №1. Створити на сторінці div, вказуємо йому якийсь виразний id, зробити йому такі стилі, щоб він виглядав як чорний квадрат 100х100 px. Додати на сторінку кнопку, по кліку на цю кнопку блок повинен зникнути. Зникання потрібно зробити усіма трьома способами (окрема кнопка під кожен спосіб):

  1. CSS (display: none), тобто просто навісити стиль на елемент
  2. JS (видалення елементу)
  3. CSS+JS (навішувати на елемент клас hidden, а в CSS прописати такі стилі, щоб вони приховували все з класом hidden)

Завдання №2. Додати кнопку і зробити так, щоб по кліку на кнопку блок пропадав, якщо він є, і щоб зʼявлявся, якщо він був прихованим (спосіб CSS+JS).

Завдання 3. Зробити на сторінці 5 таких блоків, дати їм всім однаковий клас, завдяки якому вони всі почнуть виглядати як чорні квадрати. Зробити так, щоб вони всі одночасно приховувались/зʼявлялись по натиску на кнопку.

Завдання 4. Додати на сторінку інпут, в який можна ввести css selector (query selector) і при натиску на кнопку будуть приховувались/зʼявлялись ті елементи, які відповідають цьому селектору.
